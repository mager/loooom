# loooom

Add skills from [loooom.xyz](https://loooom.xyz) to your project.

## Usage

```bash
# Add a skill to your project
npx loooom add mager/code-review

# This creates:
# skills/code-review/SKILL.md
```

## Commands

| Command | Description |
|---------|-------------|
| `npx loooom add <author>/<skill>` | Download a skill into `skills/` |
| `npx loooom search <query>` | Search for skills (coming soon) |
| `npx loooom help` | Show help |

## Requirements

- Node.js 18+
